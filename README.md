# RecipeApp
Simple and fast recipe, on-the-go!

## Synopsis

This app, RecipeApp helps to store recipes that user want to do. Hence, **RecipeApp** have 6 major features!
1. Create Recipe
2. Edit Recipe
3. Delete Recipe
4. View Recipe
5. List All Recipe
6. Filter Recipe
